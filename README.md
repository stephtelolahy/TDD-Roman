#  Roman Numerals Kata


Write a method `String convert(int)` that takes a number and converts it to the according roman representation.

Examples

   1 ➔ I
   
   2 ➔ II
   
   3 ➔ III
   
   4 ➔ IV
   
   5 ➔ V
   
   9 ➔ IX
   
  21 ➔ XXI
  
  50 ➔ L
  
 100 ➔ C
 
 500 ➔ D
 
1000 ➔ M


### Test sequences
- 1
- 2 (3)
- 5
- 6 (7, 8)
- 10
